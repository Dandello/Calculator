Данный проект представляет собой программу для арифметических вычислений и включает в себя следующие классы:
1) Calculator - основной класса проекта (содержит метод main);
2) ExprSolver - класс, отвечающий за получение значения введенного пользователем выражения;
3) View - отвечает за реализацию интерфейса программы. Состоит из поля ввода арифметического выражения и поля
вывода результата и последовательности выполняемых действий;
4) Controller - передает данные для вывода в экземппляр класса View и принимает входное арифметическое 
выражение, за обработку которого отвечает отдельный поток;
5) ResultData - класс, используемый для обмена данными между указанными выше классами. ExprSolver формирует
соответствующий объекта класса с указанием результата и истории операций и передает его Controller, который в свою
очередь отправляет его объекту View;
6) ComputationListener - класс, реализующий реакцию на событие (нажатие Enter в полее ввода выражения);
7) EvaluationThread - отвечает за реализацию потока, который вычисляет значение выражения.
